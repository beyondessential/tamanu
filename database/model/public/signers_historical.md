{% docs table__signers_historical %}
Historical VDS-NC signer records retained for compliance. Private keys have been
cleared. The table is renamed from `signers` and is no longer actively used.

See also the `certifiable_vaccines` table.
{% enddocs %}

{% enddocs %}

{% enddocs %}

{% docs signers_historical__public_key %}
Public key data.
{% enddocs %}

{% docs signers_historical__request %}
Certificate Signing Request.
{% enddocs %}

{% docs signers_historical__certificate %}
Signed Certificate (signed by the Health CA).
{% enddocs %}

{% docs signers_historical__validity_period_start %}
Before that date, the key cannot be used, and signatures generated by it cannot be trusted.
{% enddocs %}

{% docs signers_historical__validity_period_end %}
After that date, the key cannot be used, and signatures generated by it cannot be trusted.
{% enddocs %}

{% docs signers_historical__signatures_issued %}
Counter of digital signatures issued from this key.
{% enddocs %}

{% docs signers_historical__request_sent_at %}
Date at which the CSR was sent to be signed by the Health CA.
{% enddocs %}

{% docs signers_historical__working_period_start %}
Before that date, the key cannot be used.
{% enddocs %}

{% docs signers_historical__working_period_end %}
After that date, the key cannot be used.

After that date, the private key may also be nulled.

This is always less than or equal to `validity_period_end`.
{% enddocs %}
