Workhorse product development flow

The happy path from a freshly created card to shipped, verified work. Each phase pairs a collaboration mode with the skills that drive it, and changes auto-commit on every turn.

Cross-cutting

  • Auto-commit Every agent turn and user edit commits to the card's branch — no manual saves.
  • Capability handoff When the agent hits a tool limit, it offers a handoff prompt rather than looping.
    Handoff
  • Portfolio planning Cards across the project can be batched, sequenced, and reshuffled.
    Plan cards
  • Spec explorer Merged specs from the main branch form a navigable hierarchy that informs every future card.
