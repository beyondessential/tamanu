Suggestions

route: /v1/suggestions

Used by the desktop app Suggester class to add auto completing search functionality to form fields.

User

route: /v1/user

Query user data

Patient

route: /v1/patient

Query patient data, including all models with patient relations i.e. GET all patient referrals

Encounter

route: /v1/encounter

Query encounter data, including all models with encounter relations. i.e. GET all encounter diagnoses

Survey

route: /v1/survey

Query and save survey data, this includes referrals.
